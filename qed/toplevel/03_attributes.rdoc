= TOPLEVEL Annotated Attributes

Load the attributes library.

  require 'anise/attribute'

Including Anise::Attribute at the toplevel (same as Object).

  class ::Object
    include Anise::Attribute
  end

Create a class that uses it.

  class X
    attr :a, :count=>1
  end

  X.ann(:a, :count) #=> 1
